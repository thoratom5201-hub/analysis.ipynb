# Sales Performance & Customer Insights Dashboard

This project analyzes sales data to understand customer purchasing behavior, regional performance, and product category trends.

## Objective
To help businesses identify:
- Best performing regions
- High revenue product categories
- Customer buying patterns

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Electronics generate the highest revenue
- West and North regions show strong sales
- Laptops and mobiles are top selling products

## Conclusion
This analysis can help management take data-driven decisions for marketing and inventory planning.
